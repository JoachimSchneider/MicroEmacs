redraw-display

Default bindings: M-^L and M-!

Syntax:

n redraw-display

If a non zero numeric argument is specified, this command scrolls the text in the current window so that the current line is displayed as the nth line from the top of the window if n is positive, or as the -nth line from the bottom of the window if n is negative.

If no numeric argument is specified, or if n is zero, the current line is displayed at the center of the window.
