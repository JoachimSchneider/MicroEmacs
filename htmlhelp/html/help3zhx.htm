Buffer submenu

Nextinvokes the next-buffer command

Select :invokes the select-buffer command

Unmarkinvokes the unmark-buffer command

Rename :invokes the name-buffer command

Delete :invokes the delete-buffer command

Narrow to regioninvokes the narrow-to-region command

Widen from regioninvokes the widen-from-region command

Listinvokes the list-buffers command
