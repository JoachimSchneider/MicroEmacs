Compiling and installing MicroEMACS on VMS:
===========================================

- You may use the DCL script memake.com to build MicroEMACS.

- Create a directory for the MicroEMACS binary

  $ show logical sys$login
    "SYS$LOGIN" = "QDATA:[USERS.JOACHIM]" (LNM$JOB_83A28180)
  $ create /directory QDATA:[USERS.JOACHIM.BIN]

  Copy the MESMG.EXE into this directory.

  Create a foreign command `me' to call MicroEmacs:

  $ me :== $ QDATA:[USERS.JOACHIM.BIN]MESMG.EXE

- Compile also tools/scancode via `$ @mk_scancode'

- Place emacs.rc in your home directory (SYS$LOGIN).

- The keyboard may be configured via

  `set $palette KEYMAP <keybinding> <escape-sequence>'

  in emacs.rc. Use tools/scancode to find the charcter sequences
  generated by a keyboard key.

- Create a directory for the emacs cmd files and set the job-level
  logical MICROEMACS$LIB to this directory:

  $ show logical sys$login
    "SYS$LOGIN" = "QDATA:[USERS.JOACHIM]" (LNM$JOB_83A28180)
  $ create /directory QDATA:[USERS.JOACHIM.UECMD]
  $ def /JOB MICROEMACS$LIB   QDATA:[USERS.JOACHIM.UECMD]
  $ show logical MICROEMACS$LIB
   "MICROEMACS$LIB" = "QDATA:[USERS.JOACHIM.UECMD]" (LNM$JOB_83A28180)

  After copying the *.cmd files into this directory:

  $ set def MICROEMACS$LIB:
  $ dir

  Directory QDATA:[USERS.JOACHIM.UECMD]

  BPAGE.CMD;1         CPAGE.CMD;1         CUA.CMD;1           DEV.CMD;1
  EHELP.CMD;1         EHELP1.TXT;1        EHELP2.TXT;1        EMACS.RC;1
  EPAGE.CMD;1         ERROR.CMD;1         FILTER.CMD;1        LPAGE.CMD;1
  MDI.CMD;1           MEWIN.CMD;1         NEWPAGE.CMD;1       OPAGE.CMD;1
  PPAGE.CMD;1         SHELL.CMD;1         WPAGE.CMD;1

  Total of 19 files.
  $

- Add the commands

  $ me :== $ QDATA:[USERS.JOACHIM.BIN]MESMG.EXE
  $ def /JOB MICROEMACS$LIB   QDATA:[USERS.JOACHIM.UECMD]

  to your login.com.


Of course it is possible to do a system-wide install of MicroEMACS in a
similar way. Then create and populate directories like QDATA:[BIN] and
QDATA:[LIB.UECMD] and place commands like

  $ me :== $ QDATA:[BIN]MESMG.EXE

into SYS$MANAGER:SYLOGIN and

  $ def /SYSTEM MICROEMACS$LIB   QDATA:[LIB.UECMD]

into SYS$MANAGER:SYSTARTUP_VMS.COM
