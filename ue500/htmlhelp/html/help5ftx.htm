ASAVE Mode

When this mode is on, MicroEMACS automatically saves the contents of your current buffer to disk every time you have typed 256 characters. The buffer is saved to the file named on the mode line of the buffer. This mode assures you that you will loose very little text should your computer crash while you are editing. Be sure you are willing to have your original file replaced automatically before you add this mode.

The frequency of saving can be altered by changing the contents of the $asave variable.Use the set (^XA) command like this:

^XA $asave 2048

to tell MicroEMACS to automatically save the current buffer after 2048 characters are typed.

Note:the $acount variable contains the count down to the next auto-save.
