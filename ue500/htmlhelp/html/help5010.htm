The Outside World

The following commands let you interact with the Operating System or with other applications:

^X^Cexit-emacsterminates MicroEMACS

M-Zquick-exitsame as above, but saves all changed buffers first

^X!shell-commandexecutes a program within an Operating System "shell"

^X$execute-programlaunches another application

^X@pipe-commandpipes a program's output into a buffer

^X#filter-bufferfilters a buffer through a program

^XCi-shellopens an Operating System "shell"
