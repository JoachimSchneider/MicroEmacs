Getting at Files

The way you edit a file within MicroEMACS is by first reading it into a buffer, altering it and then saving it. Therefore, the most commonly used commands to access files are:

^X^Ffind-fileto read a file from disk for editing

^X^Ssave-fileto save an edited file to disk

Other read commands are:

^X^Iinsert-fileto insert at the point

^X^Rread-fileto replace the whole buffer contents

^X^Vview-fileto read for viewing, preventing any alterations

To save a buffer to another file than the one MicroEMACS would normally access, use:

^X^Wwrite-fileto overwrite the file's previous contents

^X^Aappend-fileto append to the end of the file
